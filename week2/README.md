-- Lazarus language compiler --
Building a programming language course. 1.12.23-26.01.24

Prerequisites:
1. Install Lua 5.1 or newer
2. Install LuaRocks
3. Install LPeg
4. (Optional) To run test cases, install luaunit

1. Run test cases
>lua test_*.lua

2. Compile and run Lazarus source code
> ./lazarus <filename>.lz
